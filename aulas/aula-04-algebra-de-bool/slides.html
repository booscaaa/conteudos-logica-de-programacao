Quem foi George Boole?

1815
–
1864

Matemático britânico (1815–1864) que criou um sistema onde só existem dois valores.

VERDADEIRO (1) FALSO (0)

Esse sistema é chamado de Álgebra de Bool e é a base de todos os computadores modernos.

Resumão menu_book

AND
Verdadeiro só quando todos.
1 AND 1 = 1
1 AND 0 = 0
OR
Verdadeiro quando um ou mais.
1 OR 0 = 1
0 OR 0 = 0
NOT
Inverte sempre.
NOT 1 = 0
NOT 0 = 1
