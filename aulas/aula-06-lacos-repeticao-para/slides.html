Contexto

Por que usar laços?

Imagine que você precisa mostrar os números de 1 até 100 na tela. Sem laços, seriam 100 linhas de código. Com laços, apenas 3.

cancel
Sem laço — 100 linhas
escreva(1)
escreva(2)
escreva(3)
... (97 linhas iguais)
escreva(100)
check_circle
Com laço — 4 linhas
para(inteiro i = 1; i <= 100; i++) {
    escreva(i)
}
💡 Laços permitem executar o mesmo bloco de código muitas vezes sem precisar repetir o código manualmente.
Conceito Base

O que é o comando para?

para
palavra-chave
∞→N
repetições controladas

Use o para quando você sabe exatamente quantas vezes quer repetir algo. Ele conta automaticamente — você só define o início e o fim!

lightbulb  Pense assim: "repita do número X até o número Y, contando de Z em Z"
Estrutura

Anatomia do para

para(inteiro i = 1; i <= 10; i++) {
    // bloco de código repetido aqui
}
🔢 Variável — i
O contador do laço, declarado DENTRO do para. Vai sendo atualizado a cada volta.
▶ = 1 — Início
O valor inicial do contador. É aqui que o laço começa a contar.
⏹ i <= 10 — Condição
Enquanto verdadeira, o laço continua. Quando falsa, o laço encerra.
⚡ i++ — Incremento
Executa ao final de cada iteração. Atualiza o contador antes de testar a condição novamente.

Variações

Personalizando o passo

Comando em Portugol Valores de i O que faz?
para(inteiro i = 1; i <= 5; i++) 1, 2, 3, 4, 5 Conta de 1 em 1 (padrão)
para(inteiro i = 0; i <= 10; i = i + 2) 0, 2, 4, 6, 8, 10 Conta de 2 em 2 (números pares)
para(inteiro i = 5; i >= 1; i--) 5, 4, 3, 2, 1 Contagem regressiva!
para(inteiro i = 1; i <= 9; i = i + 3) 1, 4, 7 Pula de 3 em 3
⚠️ Com condição decrescente (i >= 1) e i--, a ordem é invertida.

Aplicação Real

O acumulador: somando com laços

programa { funcao inicio() { inteiro soma = 0 // inicia em zero! para(inteiro i = 1; i <= 5; i++) { soma = soma + i // acumula } escreva("Total: ", soma) } }

O que você aprendeu hoje
