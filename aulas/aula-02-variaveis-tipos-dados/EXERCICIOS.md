# Exercícios — Variáveis, Tipos de Dados e Operações Matemáticas

---

## Variáveis e Tipos de Dados

**1.** Declare uma variável de cada tipo (`inteiro`, `real`, `caracter`, `cadeia`, `logico`), atribua valores a elas e exiba todas na tela com uma descrição.

**2.** Declare três variáveis do tipo `inteiro` sem valor inicial. Depois atribua valores a elas e exiba o resultado.

**3.** Crie um programa que armazene as seguintes informações de uma pessoa e exiba tudo formatado:
- Nome completo (`cadeia`)
- Idade (`inteiro`)
- Altura em metros (`real`)
- Peso em kg (`real`)
- Possui habilitação (`logico`)

**4.** Declare duas variáveis `cadeia`: uma com seu primeiro nome e outra com seu sobrenome. Exiba as duas juntas na mesma linha.

**5.** Crie uma variável `real` com o valor `10` e uma variável `inteiro` com o valor `3`. Divida os dois e exiba o resultado com casas decimais.

**6.** Declare duas variáveis `inteiro` com valores distintos. Troque os valores entre elas usando uma terceira variável auxiliar e exiba o resultado antes e depois da troca.

**7.** Declare uma variável `inteiro` com sua idade. Calcule e exiba quantos dias você já viveu (considere 365 dias por ano).

**8.** Declare uma variável `real` para o preço de um produto e uma variável `inteiro` para a quantidade. Calcule e exiba o valor total da compra.

**9.** Declare uma variável `cadeia` com seu nome. Exiba ela três vezes, cada uma em uma linha diferente, sem declarar novas variáveis.

**10.** Declare duas variáveis `real` representando duas notas. Calcule e exiba a diferença entre a maior e a menor, sabendo de antemão qual é qual.

## Operações Matemáticas Básicas

**11.** Peça ao usuário dois números inteiros e exiba: a soma, a subtração, a multiplicação, a divisão e o resto da divisão entre eles.

**12.** Calcule a área e o perímetro de um retângulo. O usuário informa a largura e o comprimento.

```
Área      = largura * comprimento
Perímetro = 2 * (largura + comprimento)
```

**13.** Converta uma temperatura de Celsius para Fahrenheit e para Kelvin. O usuário informa o valor em Celsius.

```
Fahrenheit = (Celsius * 9.0 / 5.0) + 32
Kelvin     = Celsius + 273.15
```

**14.** Calcule a média aritmética de 4 notas digitadas pelo usuário. Exiba cada nota e a média final.

**15.** Um produto custa R$ X e tem Y% de desconto. O usuário informa o preço e o percentual. Exiba o valor do desconto e o preço final.

```
Desconto    = preco * (percentual / 100.0)
Preço final = preco - desconto
```

**16.** O usuário digita um número. Exiba o dobro, o triplo e a metade dele.

**17.** O usuário informa seu salário atual e o percentual de aumento. Calcule e exiba o valor do aumento e o novo salário.

```
Aumento      = salario * (percentual / 100.0)
Novo salário = salario + aumento
```

**18.** Calcule a velocidade média de um carro. O usuário informa a distância percorrida (em km) e o tempo gasto (em horas).

```
Velocidade média = distancia / tempo
```

**19.** O usuário informa o valor de uma compra e o valor que pagou. Calcule e exiba o troco.

**20.** Calcule a área de um triângulo. O usuário informa a base e a altura.

```
Área = (base * altura) / 2.0
```

**21.** Um carro percorreu uma distância com uma quantidade de litros de combustível. O usuário informa os dois valores. Calcule e exiba o consumo médio em km/litro.

```
Consumo = distancia / litros
```

**22.** O usuário informa o valor total de uma conta de restaurante e o número de pessoas. Calcule quanto cada pessoa deve pagar, com e sem uma gorjeta de 10%.

```
Por pessoa       = total / pessoas
Com gorjeta      = total * 1.1 / pessoas
```

**23.** Converta uma medida em metros para centímetros, milímetros e quilômetros. O usuário informa o valor em metros.

```
Centímetros  = metros * 100
Milímetros   = metros * 1000
Quilômetros  = metros / 1000.0
```

**24.** O usuário informa o ano de nascimento de duas pessoas. Calcule e exiba a idade de cada uma (use o ano atual) e a diferença de idade entre elas.

**25.** O usuário informa o valor total de uma compra parcelada e o número de parcelas. Exiba o valor de cada parcela e o valor total com juros simples de 2% ao mês.

```
Parcela sem juros = total / parcelas
Total com juros   = total * (1 + 0.02 * parcelas)
Parcela com juros = total com juros / parcelas
```
