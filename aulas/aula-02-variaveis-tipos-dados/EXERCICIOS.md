## Variaveis e Tipos de Dados

**1.** Declare uma variavel de cada tipo (`inteiro`, `real`, `caracter`, `cadeia`, `logico`), atribua valores a elas e exiba todas na tela com uma descricao.

**2.** Declare tres variaveis do tipo `inteiro` sem valor inicial. Depois atribua valores a elas e exiba o resultado.

**3.** Crie um programa que armazene as seguintes informacoes de uma pessoa e exiba tudo formatado:
- Nome completo (`cadeia`)
- Idade (`inteiro`)
- Altura em metros (`real`)
- Peso em kg (`real`)
- Possui habilitacao (`logico`)

**4.** Declare duas variaveis `cadeia`: uma com seu primeiro nome e outra com seu sobrenome. Exiba as duas juntas na mesma linha.

**5.** Crie uma variavel `real` com o valor `10` e uma variavel `inteiro` com o valor `3`. Use casting para dividir os dois e exibir o resultado com casas decimais.

**6.** Declare duas variaveis `inteiro` com valores distintos. Troque os valores entre elas usando uma terceira variavel auxiliar e exiba o resultado antes e depois da troca.

**7.** Declare uma variavel `inteiro` com sua idade. Calcule e exiba quantos dias voce ja viveu (considere 365 dias por ano).

**8.** Declare uma variavel `real` para o preco de um produto e uma variavel `inteiro` para a quantidade. Calcule e exiba o valor total da compra.

**9.** Declare uma variavel `cadeia` com seu nome. Exiba ela tres vezes, cada uma em uma linha diferente, sem declarar novas variaveis.

**10.** Declare duas variaveis `real` representando duas notas. Calcule e exiba a diferenca entre a maior e a menor, sabendo de antemao qual e qual.

## Operacoes Matematicas Basicas

**11.** Peca ao usuario dois numeros inteiros e exiba: a soma, a subtracao, a multiplicacao, a divisao e o resto da divisao entre eles.

**12.** Calcule a area e o perimetro de um retangulo. O usuario informa a largura e o comprimento.

```
Area      = largura * comprimento
Perimetro = 2 * (largura + comprimento)
```

**13.** Converta uma temperatura de Celsius para Fahrenheit e para Kelvin. O usuario informa o valor em Celsius.

```
Fahrenheit = (Celsius * 9.0 / 5.0) + 32
Kelvin     = Celsius + 273.15
```

**14.** Calcule a media aritmetica de 4 notas digitadas pelo usuario. Exiba cada nota e a media final.

**15.** Um produto custa R$ X e tem Y% de desconto. O usuario informa o preco e o percentual. Exiba o valor do desconto e o preco final.

```
Desconto    = preco * (percentual / 100.0)
Preco final = preco - desconto
```

**16.** O usuario digita um numero. Exiba o dobro, o triplo e a metade dele.

**17.** O usuario informa seu salario atual e o percentual de aumento. Calcule e exiba o valor do aumento e o novo salario.

```
Aumento      = salario * (percentual / 100.0)
Novo salario = salario + aumento
```

**18.** Calcule a velocidade media de um carro. O usuario informa a distancia percorrida (em km) e o tempo gasto (em horas).

```
Velocidade media = distancia / tempo
```

**19.** O usuario informa o valor de uma compra e o valor que pagou. Calcule e exiba o troco.

**20.** Calcule a area de um triangulo. O usuario informa a base e a altura.

```
Area = (base * altura) / 2.0
```

**21.** Um carro percorreu uma distancia com uma quantidade de litros de combustivel. O usuario informa os dois valores. Calcule e exiba o consumo medio em km/litro.

```
Consumo = distancia / litros
```

**22.** O usuario informa o valor total de uma conta de restaurante e o numero de pessoas. Calcule quanto cada pessoa deve pagar, com e sem uma gorjeta de 10%.

```
Por pessoa       = total / pessoas
Com gorjeta      = total * 1.1 / pessoas
```

**23.** Converta uma medida em metros para centimetros, milimetros e quilometros. O usuario informa o valor em metros.

```
Centimetros  = metros * 100
Milimetros   = metros * 1000
Quilometros  = metros / 1000.0
```

**24.** O usuario informa o ano de nascimento de duas pessoas. Calcule e exiba a idade de cada uma (use o ano atual) e a diferenca de idade entre elas.

**25.** O usuario informa o valor total de uma compra parcelada e o numero de parcelas. Exiba o valor de cada parcela e o valor total com juros simples de 2% ao mes.

```
Parcela sem juros = total / parcelas
Total com juros   = total * (1 + 0.02 * parcelas)
Parcela com juros = total com juros / parcelas
```
