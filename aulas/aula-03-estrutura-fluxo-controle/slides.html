Primeiro Tipo

SE Simples

Estrutura em Portugol:

se (condição) { // executa SOMENTE // se condição = verdadeiro } // aqui o programa continua // independente do resultado
Regra: se a condição for falsa, o bloco inteiro é ignorado.

Fluxo de execução:

play_circle início
condição?
check verdadeiro executa o bloco
close falso pula o bloco
stop_circle continua aqui

Segundo Tipo

SE … SENÃO

Estrutura em Portugol:

se (condição) { // executa se VERDADEIRO } senao { // executa se FALSO }
Regra: sempre executa exatamente um dos dois blocos. Nunca os dois, nunca nenhum.

Fluxo de execução:

play_circle
condição?
check verdadeiro bloco SE
close falso bloco SENÃO
stop_circle continua aqui
SENÃO — Analogia

Par ou Ímpar

Um número sempre é par ou ímpar — nunca os dois, nunca nenhum.
O senão funciona exatamente assim.

looks_two número % 2 == 0

É PAR check_circle

| ou |
looks_one número % 2 != 0

É ÍMPAR cancel

Com SE…SENÃO, você sempre cobre os dois casos. Nenhuma situação fica sem resposta.

Terceiro Tipo

SENÃO SE
Múltiplas Condições

Estrutura em Portugol:

se (condição 1) { // executa se condição 1 = true } senao se (condição 2) { // executa se condição 2 = true } senao se (condição 3) { // executa se condição 3 = true } senao { // executa se NENHUMA for true }
1
Avalia de cima para baixo

O programa testa cada condição na ordem em que aparecem.

2
Para na primeira verdadeira

Encontrou verdadeiro? Executa o bloco e ignora o resto.

3
SENÃO é o caso padrão

Se nenhuma condição for verdadeira, cai no bloco senao.

Conceito por Nota

A professora classifica a nota do aluno em letras.
Ela verifica do maior para o menor — para na primeira que encaixar.

nota ≥ 9.0? east Conceito A — Excelente!
nota ≥ 7.0? east Conceito B — Bom!
nota ≥ 5.0? east Conceito C — Regular
senão east Conceito D — Reprovado
Nota 7.5? Falha em "≥ 9.0", passa em "≥ 7.0" → imprime Conceito B e para.

Guia Prático

Quando usar cada um?

SE
Use quando a ação é opcional.
Não fazer nada também é uma opção válida.
Exemplo: "Se tiver desconto, aplicar."
SE/SENÃO
Use quando há exatamente 2 casos e um sempre acontece.
Exemplo: "Aprovado ou Reprovado."
SENÃO SE
Use quando há 3 ou mais casos possíveis e mutuamente exclusivos.
Exemplo: "Criança / Adolescente / Adulto / Idoso."
lightbulb Dica: comece sempre pensando em quantos casos diferentes existem.
